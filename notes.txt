Serverová část
1. instalace express
    a. www stránky https://expressjs.com/
    b. možno vytvořit automatickou strukturu příkazem: $ npx express-generator
        -> strukturu zatím používat nebudu
    c. instalovat do serveru
2. instalace react
    a. www stránky https://react.dev/
    b. instalovat do klienta
3. založit GitHub
4. nastartovat server pro express
    a. jít do složky 2fish/server/
    b. příkaz node app.js
5. nastartovat react
    a. vlézt do složky 2fish/client
    b. příkaz npm start

Poznámky:
nutno používat minimální css pro bootstrap: import 'bootstrap/dist/css/bootstrap.css';
